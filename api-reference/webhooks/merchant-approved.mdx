## üìã Event Overview

**Event Type**: `MERCHANT_APPROVED`  
**Category**: Onboarding  
**Description**: Merchant account has been approved and activated

This webhook is triggered when a merchant account is approved and can start collecting payments through EximPe.

---

## üìä Payload Schema

```json
{
    "event_type": "MERCHANT_APPROVED",
    "event_time": "2024-02-15 16:53:15",
    "version": "1.0",
    "data": {
        "approved_at": "2025-06-23T07:29:29.679970Z",
        "merchant_id": "1380229823",
        "merchant_legal_name": "Herrin",
        "merchant_trade_name": "sdmndks"
    }
}
```

## üõ†Ô∏è Implementation Examples

### JavaScript/Node.js

```javascript
app.post('/webhook', (req, res) => {
    const { event_type, data } = req.body;
    
    if (event_type === 'MERCHANT_APPROVED') {
        const {
            approved_at,
            merchant_id,
            merchant_legal_name,
            merchant_trade_name
        } = data;
        
        // Handle merchant approval
        console.log(`Merchant ${merchant_id} approved at ${approved_at}`);
        console.log(`Legal Name: ${merchant_legal_name}`);
        console.log(`Trade Name: ${merchant_trade_name}`);
        
        // Update your system to enable payments for this merchant
        await enableMerchantPayments(merchant_id);
        await updateMerchantStatus(merchant_id, 'approved');
        await notifyTeam(merchant_id, 'approved');
    }
    
    res.status(200).send('OK');
});
```

### Python

```python
@app.route('/webhook', methods=['POST'])
def webhook_handler():
    data = request.get_json()
    event_type = data.get('event_type')
    
    if event_type == 'MERCHANT_APPROVED':
        webhook_data = data.get('data')
        
        approved_at = webhook_data.get('approved_at')
        merchant_id = webhook_data.get('merchant_id')
        merchant_legal_name = webhook_data.get('merchant_legal_name')
        merchant_trade_name = webhook_data.get('merchant_trade_name')
        
        # Handle merchant approval
        print(f"Merchant {merchant_id} approved at {approved_at}")
        print(f"Legal Name: {merchant_legal_name}")
        print(f"Trade Name: {merchant_trade_name}")
        
        # Update your system
        enable_merchant_payments(merchant_id)
        update_merchant_status(merchant_id, 'approved')
        notify_team(merchant_id, 'approved')
    
    return jsonify({'status': 'OK'}), 200
```

## üìã Business Logic Recommendations

When you receive this webhook, consider implementing the following:

### 1. Verify Merchant ID
```javascript
// Ensure the merchant ID matches your records
const existingMerchant = await getMerchantById(merchant_id);
if (!existingMerchant) {
    console.error(`Unknown merchant ID: ${merchant_id}`);
    return;
}
```

### 2. Update Merchant Status
```javascript
// Update merchant status in your database
await updateMerchantStatus(merchant_id, 'approved');
await updateMerchantApprovalDate(merchant_id, approved_at);
```

### 3. Enable Payment Collection
```javascript
// Enable payment collection for this merchant
await enablePaymentCollection(merchant_id);
await updateMerchantCapabilities(merchant_id, ['payments', 'settlements']);
```

### 4. Notify Relevant Teams
```javascript
// Send notifications to relevant teams
await notifyAdminTeam({
    type: 'merchant_approved',
    merchant_id,
    merchant_name: merchant_legal_name,
    approved_at
});
```

### 5. Log for Audit
```javascript
// Log the approval for audit purposes
await logAuditEvent({
    event: 'merchant_approved',
    merchant_id,
    timestamp: approved_at,
    details: {
        legal_name: merchant_legal_name,
        trade_name: merchant_trade_name
    }
});
```

## ‚ö†Ô∏è Important Notes

- **Idempotency**: This webhook may be sent multiple times. Ensure your handling logic is idempotent.
- **Verification**: Always verify the webhook signature before processing.
- **Error Handling**: Log any errors and return appropriate HTTP status codes.
- **Timing**: The `approved_at` timestamp indicates when the approval occurred, not when the webhook was sent.
