description: 'Retrieve payment details by payment ID.'
openapi: '/openapi.json GET /pg/payments/{payment_id}/'
---

<Note>
  This endpoint retrieves the complete details of an existing payment using the payment ID. You can use this to check the status and details of any payment.
</Note>

## Overview

The Get Payment endpoint allows you to retrieve complete payment information including order details, payment status, and transaction data. This is useful for payment tracking and status updates.
