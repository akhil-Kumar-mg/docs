---

## Overview

The Upload Document API allows you to upload invoice or air waybill (AWB) files for an existing order. This endpoint accepts multipart/form-data requests and supports PDF file uploads.

<RequestExample>
```bash cURL
curl -X POST "https://api.example.com/pg/orders/OD1673539082/documents/" \
  -H "accept: application/json" \
  -H "content-type: multipart/form-data" \
  -H "x-client-id: YOUR_CLIENT_ID" \
  -H "x-client-secret: YOUR_CLIENT_SECRET" \
  -F "identifier=INV1236493274" \
  -F "document_type=invoice" \
  -F "file=@invoice.pdf"
```
</RequestExample>

## Headers

<ParamField header="accept" required>
  Must be `application/json`
</ParamField>

<ParamField header="content-type" required>
  Must be `multipart/form-data`
</ParamField>

<ParamField header="x-client-id" required>
  Your API client ID
</ParamField>

<ParamField header="x-client-secret" required>
  Your API client secret
</ParamField>

## Path Parameters

<ParamField path="order_id" type="string" required>
  The order ID (e.g., OD1673539082)
</ParamField>

## Request Body

<ParamField body="identifier" type="string" required>
  Document identifier (e.g., invoice number or AWB number)
</ParamField>

<ParamField body="document_type" type="string" required>
  Type of document (`invoice` or `awb`)
</ParamField>

<ParamField body="file" type="file" required>
  The file to upload (PDF format)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<ResponseField name="data" type="object">
  Contains the uploaded document details

  <Expandable title="Show child attributes">
    <ResponseField name="document_id" type="string">
      Unique identifier for the uploaded document
    </ResponseField>

    <ResponseField name="identifier" type="string">
      Document identifier (invoice number or AWB number)
    </ResponseField>

    <ResponseField name="document_type" type="string">
      Type of document (`invoice` or `awb`)
    </ResponseField>

    <ResponseField name="file_url" type="string">
      URL to access the uploaded file
    </ResponseField>

    <ResponseField name="uploaded_at" type="string">
      Timestamp when the document was uploaded
    </ResponseField>
  </Expandable>
</ResponseField>
