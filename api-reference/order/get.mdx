---
title: 'Get Order'
api: 'GET /pg/orders/{order_id}'
description: 'Retrieve the details of an existing order.'
---

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.eximpe.com/pg/orders/OD1673539082 \
  --header 'accept: application/json' \
  --header 'x-client-id: YOUR_CLIENT_ID' \
  --header 'x-client-secret: YOUR_CLIENT_SECRET'
```
</RequestExample>

## Headers

<ParamField header="accept" type="string" required>
  Must be `application/json`
</ParamField>

<ParamField header="x-client-id" type="string" required>
  Your API client ID
</ParamField>

<ParamField header="x-client-secret" type="string" required>
  Your API client secret
</ParamField>

## Path Parameters

<ParamField path="order_id" type="string" required>
  The order ID (e.g., OD1673539082)
</ParamField>
