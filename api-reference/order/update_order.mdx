<RequestExample>
```bash cURL
curl --request PATCH \
  --url https://api.eximpe.com/pg/orders/OD1673539082 \
  --header 'accept: application/json' \
  --header 'content-type: application/json' \
  --header 'x-client-id: YOUR_CLIENT_ID' \
  --header 'x-client-secret: YOUR_CLIENT_SECRET' \
  --data '{
    "buyer": {
        "pan_number": "ACRPZ1596M",
        "dob": "1996-11-12"
    },
    "invoice": {
        "number": "INV1236493274",
        "date": "2025-06-16"
    }
}'
```
</RequestExample>

</ParamField>

<ParamField header="content-type" type="string" required>

  Your API client ID
</ParamField>

<ParamField header="x-client-secret" type="string" required>

## Path Parameters

<ParamField path="order_id" type="string" required>

</ParamField>

## Request Body

The request body is a JSON object containing only the fields you want to update. All fields are optional.

    <ParamField body="buyer.pan_number" type="string">

    <ParamField body="buyer.dob" type="string">

    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="invoice" type="object">
  Invoice information to update

  <Expandable title="Show child attributes">
    <ParamField body="invoice.number" type="string">
      Invoice number
    </ParamField>

    <ParamField body="invoice.date" type="string">

    </ParamField>
  </Expandable>
</ParamField>
