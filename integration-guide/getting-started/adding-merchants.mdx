# Adding Merchants

As a Payment Aggregator, you can onboard and manage multiple sub-merchants through EximPe's platform. This guide explains the sub-merchant onboarding process and management features.

## Sub-Merchant Onboarding Methods

You can onboard sub-merchants through two methods:

### 1. Dashboard Bulk Upload

Upload multiple sub-merchants at once using our CSV template:

1. **Download Template**
   - Login to your dashboard
   - Navigate to Sub-Merchants section
   - Click "Bulk Upload" button
   - Download the CSV template

2. **Fill Template**
   Required fields:
   - Business Name
   - Trade Name
   - Registration Number
   - Tax ID
   - Website URL
   - Business Category
   - Contact Person Details
   - Bank Account Information

3. **Upload CSV**
   - Return to the Bulk Upload section
   - Upload your completed CSV
   - Review the validation results
   - Confirm the upload

### 2. API Integration

Programmatically onboard sub-merchants using our API:

1. **Authentication**
   - Use your API credentials
   - Include the required headers

2. **API Endpoint**
   ```
   POST /v1/sub-merchants
   ```

3. **Required Parameters**
   ```json
   {
     "business_name": "Merchant Name",
     "trade_name": "Trading Name",
     "registration_number": "REG123456",
     "tax_id": "TAX123456",
     "website_url": "https://merchant.com",
     "business_category": "retail",
     "contact_person": {
       "name": "John Doe",
       "email": "john@merchant.com",
       "phone": "+1234567890"
     },
     "bank_account": {
       "account_number": "1234567890",
       "bank_name": "Sample Bank",
       "swift_code": "SAMPLECODE"
     }
   }
   ```

## Sub-Merchant Verification

All sub-merchants must complete verification before processing payments:

1. **Document Submission**
   Required documents:
   - Business Registration
   - Tax ID Proof
   - Bank Account Proof
   - Owner ID Proof

2. **Verification Timeline**
   - Standard: 2-3 business days
   - Express: 1 business day (additional fees apply)

3. **Status Tracking**
   Monitor verification status:
   - Pending Submission
   - Under Review
   - Additional Info Required
   - Approved
   - Rejected

## Managing Sub-Merchants

### Dashboard Features

1. **View & Filter**
   - List all sub-merchants
   - Filter by status, category, or date
   - Search by name or ID

2. **Update Information**
   - Edit business details
   - Update contact information
   - Modify bank details

3. **Transaction Monitoring**
   - View transaction history
   - Monitor payment volumes
   - Track settlement status
